# Theorem dependency map

The main equality follows from two independent chains.

## Upper bound

`Topological region inequality`
+ `pair savings`
+ `forced close among 4 directions`
+ `forced intriguing among 5 circles`
→ two forbidden-clique graphs
→ `colored Zykov reduction`
→ `weighted Turan theorem`
→ `weighted blocker lemma`
→ `3 x 4 matrix theorem`
→ `sigma(D,E) <= S(n)`
→ `F <= 4 choose(n,2) + S(n) + n + 1`.

## Lower bound

`exact rational four-lollipop base`
+ `pairwise chamber stability`
+ `polynomial local four-crossing family`
+ `genericization inside a chamber`
→ four-cluster blow-up with pair totals `5,7,7,7,7,7` across clusters and `4` within clusters
→ crossing total
`4 choose(n,2) + 3 sum_{i<j} m_i m_j - 2 m_0 m_1`
→ optimize by assigning the two smallest clusters to the unique five-crossing base pair
→ `F >= 4 choose(n,2) + S(n) + n + 1`.
