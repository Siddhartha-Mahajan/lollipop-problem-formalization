# Paper Bundle

Date: 2026-05-28

Downloaded references:

- `cutler_karlsson_sloane_2026_exotic_knife.pdf`: arXiv v3 paper, "Cutting a Pancake with an Exotic Knife"; proves the 4-lollipop optimum and gives current bounds for larger `n`. Source: `https://arxiv.org/pdf/2511.15864`.
- `sloane_2025_lollipop_problem_superseded.pdf`: original Dec. 2025 lollipop challenge PDF from OEIS. This is superseded by the arXiv v3 paper but useful for the problem's history. Source: `https://oeis.org/A389624/a389624.pdf`.
- `sloane_lollipop_problem_homepage.pdf`: Sloane-hosted copy of the earlier lollipop note. Source: `https://neilsloane.com/doc/LLL10.pdf`.

Cached OEIS pages, figures, and Karlsson construction code are in `../web_sources/`.
