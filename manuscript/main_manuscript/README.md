# Lollipop formula manuscript

This package contains the revised LaTeX manuscript and a Lean 4 formalization package for the combinatorial spine of the proof.

Files:

- `main.tex` - manuscript source.
- `main.pdf` - compiled manuscript.
- `lean/` - Lean 4 source package. The geometric lollipop facts are represented as external hypotheses, matching the cited inputs in the paper; the combinatorial theorem is organized into the same Zykov, blocker, and matrix-theorem layers used in the manuscript.

The manuscript is written as an ordinary theorem-proof paper.
